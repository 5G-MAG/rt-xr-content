## scene-av-combined

### Summary
audio and video packaged as seperate tracks of a single mp4, explicit track to buffer mapping


### Extensions used

- MPEG_media
- MPEG_buffer_circular
- MPEG_accessor_timed
- MPEG_audio_spatial
- MPEG_texture_video
